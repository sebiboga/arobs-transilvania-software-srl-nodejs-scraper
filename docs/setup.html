Cerințe de Sistem

  • Node.js 20+ (recomandat 20 LTS) - Runtime JavaScript
  • npm (inclus cu Node.js) - Manager de pachete
  • Acces la internet - pentru ANAF API, Solr, iCIMS
  • Git (opțional) - pentru clonare repository
Notă: Nu este nevoie de Chrome, Docker, PowerShell sau alte servicii locale. Totul rulează direct cu Node.js.

Pași de Instalare

1. Clonează repository-ul:
git clone https://github.com/sebiboga/arobs-transilvania-software-srl-nodejs-scraper.git
cd arobs-transilvania-software-srl-nodejs-scraper
2. Instalează dependențele:
npm install

Instalează: node-fetch, cheerio, dotenv (dependencies) și jest, jest-html-reporter (devDependencies).

3. Configurează autentificarea Solr:
echo "SOLR_AUTH=solr:SolrRocks" > .env.local

Sau exportă variabila direct:

export SOLR_AUTH="solr:SolrRocks"
4. Rulează scraper-ul:
npm run scrape
5. Rulează testele (opțional):
npm test
Notă: Dacă SOLR_AUTH nu este configurat, scraper-ul va funcționa parțial (va face scraping și va salva local în tmp/jobs.json), dar nu va putea trimite date la Solr.

Configurare Mediu

Variabile de Mediu

Variabilă Descriere Obligatoriu
SOLR_AUTH Credențiale Solr în format username:password Da (pentru upload Solr)
Fișier .env.local (ignorat de git):
SOLR_AUTH=solr:SolrRocks
SECURITATE: Fișierul .env.local este inclus în .gitignore și nu va fi niciodată comis în repository. În GitHub Actions, variabila este setată ca Secret.

Verificare Instalare

Verifică Node.js:

node --version
# Ar trebui să afișeze: v20.x.x sau mai mare

Verifică npm:

npm --version
# Ar trebui să afișeze: 10.x.x sau mai mare

Verifică dependențele:

npm ls cheerio node-fetch dotenv
# Ar trebui să afișeze versiunile instalate

Verifică că scraper-ul pornește:

node --no-deprecation index.js --test
# Va rula doar prima pagină (--test) și va afișa progresul

Depanare

Eroare: "SOLR_AUTH not set in environment"

  • Creează fișierul .env.local cu conținutul: SOLR_AUTH=solr:SolrRocks
  • Sau exportă variabila: export SOLR_AUTH="solr:SolrRocks"

Eroare: "fetch failed" la iCIMS

  • Verifică conexiunea la internet
  • Site-ul iCIMS poate fi temporar indisponibil
  • Încearcă din nou după câteva minute

Eroare: "ANAF API error"

  • API-ul demoanaf.ro poate fi temporar indisponibil
  • Scraper-ul are retry logic (3 încercări)
  • Alternativ, rulează cu cache existent în tmp/company.json

Testele eșuează?

  • Rulează: npm install pentru a reinstala dependențele
  • Verifică versiunea Node.js: node --version (trebuie 20+)
  • Rulează testele individual: npm run test:unit
