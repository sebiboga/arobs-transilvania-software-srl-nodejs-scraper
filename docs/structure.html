Descriere Foldere și Fișiere

Fișier/Folder Descriere
index.js Scraper principal. Folosește node-fetch + cheerio pentru a extrage job-urile AROBS de pe platforma iCIMS. Parsează pagină cu pagină, extrage titlu, URL, locație, categorie, senioritate, skill-uri și trimite datele în Solr.
company.js Validare companie prin ANAF API și Peviitor API. Verifică statusul companiei (activ/inactiv), salvează datele în cache local (tmp/company.json) și validează modelul Company.
solr.js Modul pentru operații CRUD cu Solr (query, upsert, delete) pe core-urile job și company. Include funcții de backup și verificare URL-uri.
validate-jobs.js Validator URL-uri job. Verifică dacă URL-urile mai sunt active (200 OK) sau au expirat (404/keywords). Poate șterge job-urile expirate din Solr.
demoanaf.js Punct de intrare CLI pentru modulul ANAF. Permite căutare companii după brand sau interogare după CIF.
src/anaf.js Integrare API ANAF (demoanaf.ro). Interoghează date oficiale companie: nume, CUI, status, adresă, TVA, e-Factura. Include retry logic.
package.json Configurare proiect ESM. Include scripturi npm: npm test, npm run scrape, npm run test:e2e. Dependențe: node-fetch, cheerio, dotenv, Jest.
docs/ Documentație HTML completă pentru înțelegerea proiectului.
tests/unit/ Teste unitare Jest: index.test.js (parsare iCIMS, locații, tag-uri), company.test.js, solr.test.js, demoanaf.test.js.
tests/e2e/ Teste end-to-end care fac fetch real la iCIMS și parsează răspunsul live.
tests/integration/ Teste de integrare care verifică fluxul complet de extragere tag-uri și mapare date.
.github/workflows/ Workflow-uri GitHub Actions: scrape.yml (programat), test.yml (CI), deploy.yml (deploy).
job-model.md Schema completă a modelului Job - câmpuri, tipuri, reguli de validare.
company-model.md Schema completă a modelului Company - câmpuri, tipuri, status-uri.
webscraper/arobs.md Instrucțiuni specifice pentru scraping AROBS de pe iCIMS.

Note

  • node_modules/ - Folder generat automat. Pentru a restaura: npm install. Nu trebuie inclus în git.
  • tmp/ - Fișiere temporare generate în timpul scraping-ului (jobs.json, company.json). Ignorate de git.
  • Fișierele .md din rădăcină sunt documentație auxiliară (TOPICS, PUBLIC, files, etc.).

Fluxul de Date

ANAF API (validare companie)
    ↓
iCIMS careers page (fetch HTML)
    ↓
cheerio parse (extragere job-uri)
    ↓
mapToJobModel (tag-uri + locații + workmode)
    ↓
Solr Job Core (upsert)
    ↓
Solr Company Core (upsert lastScraped)
    ↓
Teste Jest (validare)
