Bine ai venit! 👋

Acest proiect este un scraper Node.js standalone care extrage automat locurile de muncă de pe platforma de cariere iCIMS a companiei AROBS Transilvania Software (CIF 11291045).

Scop: Colectăm job-urile AROBS de pe careers-arobs.icims.com prin HTML scraping cu cheerio și le indexăm în Apache Solr pentru agregare pe peviitor.ro.

🎯 Obiective

Sistem automatizat de colectare a job-urilor AROBS:

1. Scraping iCIMS

Extragem locurile de muncă de pe careers-arobs.icims.com/jobs/search prin parsare HTML cu cheerio.

2. Validare ANAF

Verificăm compania în registrul ANAF (CIF 11291045) pentru status activ și date oficiale.

3. Indexare Solr

Încărcăm datele structurate în Apache Solr (core-urile job și company) pentru căutare și filtrare.

4. Testare Jest

Teste unitare, de integrare și E2E cu Jest pentru validarea corectitudinii datelor.

Ce face acest proiect?

🌐 Scraping HTML

Folosim node-fetch pentru a descărca paginile iCIMS și cheerio pentru a parsa și extrage job-urile.

🏢 Validare Companie

Interogăm API-ul ANAF (via demoanaf.ro) și API-ul Peviitor pentru validarea companiei.

💾 Indexare Solr

Datele sunt trimise către Apache Solr (solr.peviitor.ro) pentru indexare și upsert.

✅ Testare Automată

Testele Jest verifică parsarea HTML, extragerea tag-urilor, maparea locațiilor și fluxurile complete.

💻 Arhitectură Node.js Standalone

Acest proiect este un scraper Node.js pur, fără dependențe de browser sau Docker.

Arhitectura:
  • Node.js 20+ - runtime JavaScript
  • cheerio - parsare HTML rapidă (fără browser)
  • node-fetch - cereri HTTP
  • Apache Solr - stocare date (solr.peviitor.ro)
  • Jest - testare automată
  • ESM Modules - import/export modern
Cerințe Sistem:
  • Node.js 20+ (recomandat 20 LTS)
  • npm (inclus cu Node.js)
  • Acces la internet (pentru ANAF API, Solr, iCIMS)
  • SOLR_AUTH configurat în variabile de mediu
🔒 Confidențialitate:
  • Datele de scraping sunt trimise direct la Solr (solr.peviitor.ro)
  • Nu se stochează date local (doar cache temporar în tmp/)
  • Autentificarea Solr se face prin Basic Auth
  • Nu se folosesc servicii third-party pentru procesare
